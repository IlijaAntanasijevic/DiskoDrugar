# Disko Drugar - prototip

Staticki prototip sajta i admin CMS-a. Sve stranice su klikabilne; jedinstvene stvari (blog clanak, detalj putovanja) vode na postojece primer-stranice.

## Deploy

**Vercel:** prevucite ovaj folder na vercel.com/new (ili `vercel deploy`). Nikakva konfiguracija nije potrebna.

**GitHub Pages:** push-ujte sadrzaj foldera u repo, pa Settings -> Pages -> Deploy from branch (root).

Ulazna tacka: `index.html` (pocetna). Admin: `admin-dashboard.html` (link "Admin" je u gornjoj traci pocetne).

Napomena: stranicama je potreban internet (Google fontovi, React CDN, stock fotografije sa Unsplash-a).
